# Kotlin GitHub Client

This project is a mobile application built with Kotlin. It uses GitHub APIs to search for users and show the selected user's repositories, followers, and following.

## Used Technologies

- **Retrofit**
- **RxJava**
- **Navigation**
- **Glide**
